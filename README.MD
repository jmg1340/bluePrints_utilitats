## App UTILITATS

Requeriments **linux** (`apt-get`):
- python3
- sshpass

<br/>

Creacio d'entorn virtual (`python3 -m venv <carpeta>`) i activació (`<carpeta>/bin/activate`) per instal·lar **llibreries python** (`pip`):
- flask
- flask-socketIO
  
<br>

**Variables d'entorn** (`export variable=`):
- APP_SETTINGS_MODULE
- PWDASE
- PWDCOS
